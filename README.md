## Internal users
Only for researchers that are part of the project.

### Install necessary software

**git**

`install git`

**mamba or conda**

`install mamba`

### Clone repository & create environment
Only need to be done once.
```sh
$ git clone ...
```
```sh
$ cd ...
```
```sh
$ mamba create ...
```

### Open folder & activate environment
```sh
$ cd ...
```
```sh
$ mamba activate ...
```
